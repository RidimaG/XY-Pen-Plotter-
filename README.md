# XY-Pen-Plotter-
The XY Pen Plotter is a custom-built CNC-style machine that translates digital coordinates into physical drawings. Built on an ESP32 architecture, it utilizes a mobile interface for real-time control and wireless data transmission, bridging the gap between digital design and physical execution.
